spa-tools

Single-file HTML tools — no build step, no framework, just vanilla JS.

  • EPUB Preparer — Edit EPUB metadata, strip Project Gutenberg boilerplate, and download the corrected file.
  • F1 Race Tracker — Formula 1 race timing tower with live gaps, intervals, tyre data, and race control feed powered by OpenF1.
  • F1 Tracker — Live Formula 1 countdown, circuit weather, standings, and news — all in one page.
