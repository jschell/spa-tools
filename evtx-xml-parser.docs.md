This tool parses raw Windows Event Log XML and presents it as a clean, annotated three-panel view. Paste a full `<Event>` element or a bare `<EventData>` fragment into the text area and click Parse Event (or press Ctrl+Enter). If the XML contains multiple `<Event>` elements wrapped in a container, each is shown as a numbered tab.

The System Fields panel displays provider name and GUID, EventID with a link to the Ultimate Windows Security encyclopedia entry and a short description, a colour-coded level badge (Audit Success in green, Audit Failure in red, Information in blue, Warning in orange, Error in red), channel, computer name, and human-readable timestamp alongside the raw UTC value. The EventData panel lists every field name and value in a table, with short annotations for common fields such as LogonType, SubStatus, and TicketEncryptionType. SID values get an inline Copy button; hexadecimal values are shown with their decimal equivalent; GUIDs are normalised to uppercase with braces. A triage note is displayed at the top of the System Fields panel for events with known investigation priorities.

The KQL Generator panel produces a Microsoft Sentinel query pre-filtered to the parsed EventID, with one commented-out `| where` clause per EventData field as a starting template and a `| project` line listing every field present in the event. Use the Copy KQL button to copy it to the clipboard. The tool runs entirely in the browser with no external requests and works offline after the initial page load.
